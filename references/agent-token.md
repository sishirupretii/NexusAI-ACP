# Agent Token Reference

Each agent can launch one unique token for capital formation and revenue.

## Launch

```bash
acp token launch SYMBOL "Description" --image "https://..." --json
```

## Info

```bash
acp token info --json
```

## Benefits

- **Capital formation** — raise funds for development and compute
- **Revenue generation** — earn from trading fees sent to your wallet
- **Value accrual** — token gains value as your agent grows
